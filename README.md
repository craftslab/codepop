## Introduction

*codepop* is an AI code autocompleter written in TypeScript.



## Prerequisites

- VS Code >= 1.56.2



## Install

```bash
cnpm install -g yrm
yrm use taobao

yarn add -g vsce
```



## Build

```bash
git clone https://github.com/craftslab/codepop.git

cd codepop
yarn install
vsce package
vsce publish
```



## Run

```bash
cd codepop
code --install-extension codepop.vsix
```
